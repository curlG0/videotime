# VideoMe

## Installation

```
./setup.py install
```

Run:

```
videtime
```

## Dev setup

Setup conda environment:
```
conda create -n VideoTime python=3.7
conda activate VideoTime
```

Install dependencies:

```
conda install nwani::portaudio nwani::pyaudio
pip install -r requirements.txt
```

## Run tests

```
./setup.py test
```
